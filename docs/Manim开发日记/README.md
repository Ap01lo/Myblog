这个开源引擎是由python开发的，兴趣所致，正好和同学想研究一些也来做一些小东西，但是由于时间很随缘，所以这个也是学的很随缘，但是一定会抽时间完成。

至于为什么单独开一个档出来，是因为这个没有现成的教程，所有东西几乎都要自己探索，也不会很顺利，所以在这里记录开发历程。
